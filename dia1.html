Portafolio Personal

Experiencia laboral relevante

Proyecto 1: Diseño e Implementación de un Sistema de Control para una Planta de Tratamiento de Agua

Descripción: Este proyecto consistió en el diseño e implementación de un sistema de control para una planta de tratamiento de agua, utilizando PLCs y SCADA para monitorear y controlar los procesos de tratamiento.

Proyecto 2: Mantenimiento Predictivo en una Refinería

Descripción: En este proyecto, se implementó un programa de mantenimiento predictivo en una refinería, utilizando técnicas de análisis de vibraciones y termografía para identificar posibles fallas en los equipos antes de que ocurran.

Proyecto 3: Implementación de un Sistema de Monitoreo en una Planta Química

Descripción: Este proyecto consistió en la implementación de un sistema de monitoreo en una planta química, utilizando sensores y sistemas de control para optimizar los procesos de producción.
