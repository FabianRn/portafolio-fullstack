# Portafolio Full Stack — Fabian Reyes

Portafolio profesional de **Fabian Eduardo Reyes Navarro**, Ingeniero de Automatización Industrial migrando a desarrollador Web Full Stack.

## Tecnologías

- HTML5, CSS3, JavaScript (vanilla)
- Diseño responsive con 3 breakpoints
- Modo oscuro/claro
- Animaciones con Intersection Observer
- API REST (Node.js) para perfil dinámico
- Integración con GitHub API

## Secciones

- Hero — presentación principal
- Perfil Profesional — resumen de carrera y objetivo
- Sobre mí / Experiencia — trayectoria laboral
- Timeline — camino profesional
- Proyectos — dashboards KPI y sistemas de gestión
- Habilidades — tecnologías y herramientas
- Contacto — formulario de contacto

## Uso

Abrir `index.html` en el navegador o servir con un servidor local.

```bash
npx serve .
```
